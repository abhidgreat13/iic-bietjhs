Introduction

The event is divided into two rounds.
● The first round will be a blank coding round.
- A simple coding problem will be given and students will have to code the solution.
- But computer screens will be turned off, participants will be using only the keyboard to type the solution code
- Time duration: 30 min
- Marking scheme: 100 - (number of mistakes) + Time left
● Qualified candidates of round 1 will be promoted to round 2.
● The second round will be a coding competition
- 3 to 4 coding problems will be provided to the participants.
- Participants will have to write the solutions according to the problem Statement.
- Time duration: 60 min
- Marking scheme: Each problem will have its score
- For correct submission: +50 points
- For each incorrect submission: -3 points penalty
● The top 3 Participants will be awarded.

Flow of Event


1. Firstly a workshop will be taken, in order to brief students about the coding language and the interface of the coding platform (hacker rank, code chef, etc.).
2. The workshop will be taken 5 to 6 days prior to the main event date.
3. We will start taking the registrations 2 days before the main event.
4. On the event day in the morning, we will conduct round 1 - blank coding round.
5. In the afternoon the second round- the competitive coding round will be conducted.
6. The overall score of both the events will be calculated and winners will be declared.

Who can Participate?


This event is open to all. Any team from any recognized institution or college can participate.

Rules and Regulations


1. Copying someone else’s code will lead to immediate disqualification from the whole event.
2. Discussing any problems (or any aspect of the problem) on any other platform on internet, if identified, could lead to disqualification of the participant from the contest.
3. Discipline should be maintained throughout the event.
4. Participants must coordinate with the event coordinators.
